This plugin helps you create new members in your Wishlist Membership site based on ONTRAPORT contacts and tags. It is the easiest way to Automate your Wishlist Membership Site with ONTRAPORT.

== Description ==

Each Membership Level in Wishlist relates to two tags that you will create manually in ONTRAPORT. Then you will create some Rules that are triggered by adding or removing these particular tags. The Rules ping a URL and send contact info to the Wishlist Site to Update or Create the new user and giving them the correct access based on their tags in ONTRAPORT.

== Installation ==

1. Upload `ontraport-wishlist-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
